# 品質ルール

ユーザーが実際に作る前に、よくある設計ミスを検出するための物理・工学ルールです。
バリデーター（`scripts/validate_spec.py`）がこれらを自動的に適用します。これらを把握しておくと、最初から良い spec を生成しやすくなります。

## スパン限界（棚のたわみ）

無支持スパン = 2つのサポート間の距離。板は長いスパンでたわみます。

| 材料 | 安全な無支持スパン | 備考 |
|---|---|---|
| SPF 1×4（厚み 19mm、幅 89mm） | 600mm | 標準的な荷重（本、台所用品）に対して |
| SPF 1×6（19×140） | 700mm | |
| SPF 1×8（19×184） | 750mm | |
| 合板 t=15 | 600mm | |
| 合板 t=18 | 700mm | |
| 集成材 t=25 | 900mm | |
| SPF 2×4（厚み 38mm、立て使い） | 1200mm | テーブル支持として使う棚に |

重い荷重（本、書類）の場合は 20% 減らしてください。

## 端距離 / ダボ配置

- ダボの中心は**板の端から少なくとも 30mm**
- 平行な2本のダボ：**80〜150mm 間隔**（薄い板ほど狭く、厚い板ほど広く）
- ネジの下穴：ネジ径の 60%、深さはネジ長の 2/3 以上

## ダボ穴の深さ vs 板厚

穴の深さは**板厚 − 3mm 以下**にしてください（貫通を防ぐため）。

8mm × 30mm ダボの場合：
- 19mm の板（1×4 端面）：穴の深さ最大 16mm → 15mm を使用
- 19mm の板（1×4 木口面）：同じ制限
- 38mm の板（2×4）：穴の深さ 25mm まで問題なし

## 材料の長さチェック

いずれかのパーツの最大寸法が 1820mm を超える場合、ユーザーに警告します。必要なもの：
- 2440mm の板（ホームセンターの在庫が限られる）、または
- 合板からパネルカット（常にシートとして扱う）

2440mm を超える場合、ユーザーは 3650mm の材を使って自宅で事前カットするか、2枚の板を継ぎ合わせる必要があります。

## カットレイアウトの実現可能性

- 同じ材料のパーツ長の合計 × (1 + 5% kerf ロス) ÷ 在庫長 = 必要な板の枚数
- ミスカット用に板を1枚追加する（目安）

## 接合強度の目安

| 接合方法 | 強度 | 用途 |
|---|---|---|
| ネジのみ | 引張に弱い | 隠れた接合部、試作品 |
| ネジ + 接着剤 | 中程度 | 構造的な箱の接合 |
| ダボ + 接着剤 | 中〜強 | 見える家具、フラッシュ面 |
| ポケットホール + 接着剤 | 強い | 片側が隠れる場合 |
| ほぞ組み（mortise + tenon） | 非常に強い | 長く使う家具（上級者向け） |
| 溝加工（dado） + 接着剤 | 強い | 棚板が剪断荷重を負担する本棚 |

## 木目方向のルール

長い細いパーツ（側板、棚板、縦桟）：**木目は長軸方向に沿って走る**。
広いパネル（天板、扉）：木目は最も長い方向に沿って走る。

クロスグレイン接合（木口面と木目面の接合）は弱いです。ダボや機械的ファスナーを使用してください。

## 重心 / 転倒

高さが奥行きの 2 倍以上の背の高い家具は：
- 壁に固定する、または
- 予想より広い台座を設計する、または
- 重いものを下段のみに配置する

バリデーターは高さ/奥行き比 > 2.5 のとき警告します。

## ハードウェアの耐荷重

- M6 壁アンカー（石膏ボード用アンカー）：静荷重 約 15kg
- 石膏ボードアンカー（トグル式）：約 25kg
- 下地ビス（柱固定）：ネジ1本あたり 約 50kg
- 高さ 1500mm を超える家具は必ず壁に固定する

## よくある spec のミス

1. **棚板の向きが間違っている**：250mm 深のキャビネットに「300×600」の棚板は入りません。`dimensions` が `overall` の制約に合っているか確認する。
2. **側板が短すぎる**：側板の高さは（高さ − 天板 − 底板）ではなく全体高さと同じにすること。
3. **パーツ高さの合計 ≠ 全体高さ**：内部スパン + 厚みを合計して確認する。
4. **背板を忘れている**：背板のない本棚は直角が狂います。5.5mm または 9mm の背板を推奨する。
5. **扉のクリアランスを見落としている**：蝶番付き扉は各側に 2〜3mm のクリアランスが必要。

## コスト見積もりの警告

- 材料費はプロジェクト総コストの通常 60〜70%
- ハードウェア（ネジ、ダボ、蝶番）：10〜15%
- 仕上げ（オイル、塗料、刷毛）：15〜20%
- サンドペーパー、マスキングテープ等：5%

見積もり総額が ¥3,000 未満の場合は再確認 — 見落としている項目がある可能性が高い。¥30,000 超の場合は、まとめ買いを提案する。
